# Health_Insurance_Project
This project aims at exploring if the Smoking Status and Body Mass Index (BMI) of an individual amplifies the cost charges of Health Insurance in US (done on secondary data set )


- Smoking status is identified as the primary cost driver, with significantly higher average expenses compared to non-smokers
- People with High BMI taht are also smokers represent the highest-cost segment
- The analysis Supports more accurate pricing models based on combined risk profiles
